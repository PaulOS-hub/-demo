# 网易小程序demo，使用网易api接口。
登录的时候用你自己的网易云账号密码，不然登不上。
获取的数据歌曲，都是和你网易云app的歌曲数据一样的

## 启动
1.用微信开发者工具打开
2.拉server代码，npm start跑起来
3. 预览体验

## 体验
1.可以在线听歌
2.可以看video
3.同步你的网易云app和数据
4.模拟登录小程序
5.配置接口等
